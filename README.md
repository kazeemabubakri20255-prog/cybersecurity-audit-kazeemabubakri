# Beginner Cybersecurity Audit Project

## Name
Kazeem Abubakri

## Project Title
Personal Device / Small Office Cybersecurity Audit

## Objective
The aim of this project is to identify basic cybersecurity weaknesses in a personal device, smartphone, or small network setup and suggest practical security improvements.

## Scope
I audited my personal laptop.

## Tools Used
- Device settings
- Security settings pages
- Antivirus dashboard
- System update page
- Browser security settings

## Files in this Repository
- `checklist.md` — Security checklist with pass/fail status for each item
- `findings-report.md` — Detailed report of what was discovered during the audit
- `recommendations.md` — Practical steps to fix identified security weaknesses
- `screenshots/` — Evidence screenshots taken during the audit

## Key Findings
- Password protection was enabled ✅
- Antivirus is installed and active ✅
- Firewall was active ✅
- Operating system updates were missing ⚠️
- Unused applications were still installed ⚠️
- Browser saved passwords lacked extra protection ⚠️

## Risk Level
**Medium Risk**

## Conclusion
This project helped me understand how to identify common cybersecurity risks in a personal device and recommend practical solutions to improve security. Regular audits like this one are an important habit for staying protected online.
