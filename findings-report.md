## Summary
I carried out a basic cybersecurity audit on my personal laptop to identify common security weaknesses. The audit covered password protection, antivirus status, firewall settings, operating system updates, installed applications, and browser security.

---

## Detailed Findings

### ✅ Finding 1: Strong Password Enabled
The laptop is protected with a strong login password. This is a good first line of defence against unauthorized physical access.

---

### ✅ Finding 2: Antivirus Installed and Active
An antivirus program is installed and currently running. This helps protect the device from malware, viruses, and other malicious software.

---

### ✅ Finding 3: Firewall is Enabled
The system firewall is turned on. This helps block unauthorized network connections to and from the device.

---

### ❌ Finding 4: Operating System Not Fully Updated
The operating system has pending updates that have not been installed. Outdated software may contain known security vulnerabilities that attackers can exploit.

**Risk:** High — unpatched systems are a common entry point for cyberattacks.

---

### ⚠️ Finding 5: Unused Applications Still Installed
Several applications that are no longer in use remain installed on the device. Unused apps can still contain vulnerabilities and increase the attack surface of the device.

**Risk:** Medium — every installed app is a potential security risk if not maintained.

---

### ❌ Finding 6: Browser Saved Passwords Not Protected
The browser stores saved passwords without an additional layer of security (such as a master password or password manager). Anyone with access to the browser could view or use these saved credentials.

**Risk:** High — this can lead to account compromise if the device is accessed by someone else.

---

## Risk Level

| Risk Level | Reason |
|------------|--------|
| **Medium Risk** | The device has core protections in place (password, antivirus, firewall), but missing updates and poor browser password security create exploitable weaknesses. |

---

## Conclusion
The laptop has a reasonable security baseline, but several important gaps need to be addressed. Updating the OS, removing unused apps, and securing browser passwords are the most critical next steps.
